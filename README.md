# aunzite.github.io

Personal portfolio site for Aun Ali — Computing student at Queen's University.

**Live:** [aunzite.github.io](https://aunzite.github.io)

## Stack

Vanilla HTML, CSS, and JavaScript. No frameworks, no build step.

- [Lenis](https://github.com/darkroomengineering/lenis) — smooth scroll
- [Cormorant + Inter](https://fonts.google.com) — typography
- Deployed via GitHub Pages

## Structure

```
index.html      markup
style.css       all styles
script.js       scroll, cursor, accordion
favicon.svg     4-pointed star icon
lenis.min.js    smooth scroll library
```

## Local dev

```bash
python -m http.server 5500
# open http://localhost:5500
```
